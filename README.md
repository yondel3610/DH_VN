## Installation & Running

### From Source (Development)

1. Install [Ren'Py SDK](https://www.renpy.org/latest.html)
2. Clone this repository:
   ```bash
   git clone https://github.com/yondel3610/Dragon-s-Heart-VN.git
   ```
3. Open Ren'Py Launcher
4. Click "Add Project" and navigate to the project folder
5. Select the project and click "Launch Project"

### From Terminal (VSCode)

```bash
# Windows
renpy.exe /path/to/project

# macOS
./renpy.sh /path/to/project

# Linux
./renpy.sh /path/to/project
```

## Building Distributables

1. Open Ren'Py Launcher
2. Select the project
3. Click "Build Distributions"
4. Select target platforms:
   - Windows (PC)
   - macOS
   - Linux
   - Android (requires RAPT)
   - Web (HTML5)
   - iOS (requires Xcode on Mac)

## Development Notes

### Naming Conventions

All assets follow these conventions:

- **Image tags:** `bg_location_name` or `character_expression`
- **Audio variables:** `audio.category_name`
- **Labels:** `chapter_X_scene_name`
- **Variables:** `character_affection` (snake_case)
- **Character defines:** `character_name` (lowercase)

### Save Compatibility

- Save files are stored in `%APPDATA%/RenPy/dragons_heart-[id]/`
- Do not modify save files directly
- Breaking changes between versions may break old saves
